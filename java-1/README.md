# This is part of a series of java programs written as excercises.

Person.java and Project.java and two Classes for creating objects.
  
* **Person** stores:
  * name
  * role
  * details

This person object is used to populate certain attributes of Project.

* **Project** stores
  *  projectNumber
  *  projectName
  *  buildingType
  *  adress
  *  erf
  *  totalFee
  *  totalPaid
  *  projectDeadline - *(java LocalDate object)*
  *  architect - *(Person Object)*
  *  contractor - *(Person Object)*
  *  customer - *(Person Object)*
  *  isFinalized - *(Boolean True/False)*
  *  completionDate - *(java LocalDate object)*
  *  (Contains 2 static methods)


**Class Attributes** for both Object Classes are **private** and attributes are *called via methods* from within each class.
